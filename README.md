# Thesis

Αυτό το repository περιέχει τον κώδικα και τα αρχεία που χρησιμοποιήθηκαν για την ατομική διπλωματική μου εργασία:

**Disinformation Campaign Analysis: The Case of the 2024 Olympic Games**

Η εργασία επικεντρώνεται στην ανάλυση της πόλωσης και της παραπληροφόρησης μέσα από ειδησεογραφικά άρθρα που σχετίζονται με τους Ολυμπιακούς Αγώνες του 2024. 

## Περιεχόμενα

- Python scripts που χρησιμοποιήθηκαν για την συλλογη των αρθρων και την ανάλυση
- Αποτελέσματα και υποστηρικτικό υλικό της διπλωματικής εργασίας
